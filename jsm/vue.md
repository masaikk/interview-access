**vue-cli-service封装了webpack**

例如demo1/node_modules/@vue/cli-service/package.json中，指定了bin所指向的二进制文件的位置。

```json
{
      "bin": {
    "vue-cli-service": "bin/vue-cli-service.js"
  },
}
```
